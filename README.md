# 🌱 Carbon Trace – AI-Powered Carbon Footprint Awareness Platform

Carbon Trace is an interactive sustainability platform that helps individuals understand, monitor, and reduce their carbon footprint through personalized insights, environmental analytics, gamification, and AI-powered recommendations.

Built as part of the **Hack2Skill Carbon Footprint Awareness Platform Challenge**, Carbon Trace transforms environmental awareness into actionable daily habits.

# 🌍 Problem Statement

Many people want to live more sustainably but struggle to understand:

* How their daily activities impact the environment
* Which habits contribute the most carbon emissions
* What practical actions they can take to reduce their footprint

Carbon Trace addresses this challenge by providing a simple, engaging, and data-driven experience that turns awareness into action.

---

# ✨ Key Features

## 🧮 Carbon Footprint Assessment

Users complete an interactive sustainability questionnaire covering:

* Transportation habits
* Energy consumption
* Food choices
* Waste management practices

The system generates a personalized carbon footprint profile.

---

## 🤖 AI-Powered Sustainability Insights

Carbon Trace provides:

* Personalized recommendations
* Reduction strategies
* Environmental impact explanations
* Actionable sustainability guidance

Powered by Google AI technologies.

---

## 📈 Weekly Emission Analytics

Interactive visualizations help users understand:

* Emission trends
* Transportation impact
* Energy usage
* Diet-related emissions
* Waste contribution

Built using Recharts for fast and responsive analytics.

---

## 🏆 Eco Achievement System

Gamification encourages long-term engagement through:

* Sustainability badges
* Climate milestones
* Habit streaks
* Carbon reduction achievements

Users are rewarded for adopting eco-friendly behaviors.

---

## 🎙️ Voice Assistant

Integrated browser-based voice interaction enables users to:

* Ask sustainability questions
* Receive spoken responses
* Explore environmental insights hands-free

Powered by the Web Speech API.

---

## 🎨 Theme Personalization

Multiple visual themes allow users to customize their experience while maintaining accessibility standards.

---

## 🔐 Authentication Experience

Users can:

* Create profiles
* Explore personalized dashboards
* Track sustainability progress

# ⚡ Performance Optimizations

Carbon Trace was designed with efficiency in mind.

### Memoized Analytics

Emission calculations and chart datasets are memoized to prevent unnecessary recalculations.

### Lightweight Architecture

* React + Vite
* Fast hot reload
* Optimized production builds

### Responsive Charts

Charts adapt to all screen sizes while minimizing layout shifts.

### Efficient State Management

Local state is used strategically to reduce memory overhead and unnecessary renders.

# 🧪 Development Setup

## Install Dependencies


## Run Development Server

```bash
npm run dev
```

## Build Production Version

```bash
npm run build
```

## Start Production Server

```bash
npm run start
```
